# CogDB Graph Explorer

Interactive demo of [CogDB](https://github.com/arun1729/cog) — an embedded graph database for Python.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features

- **Add Triples** — Build your graph with subject-predicate-object relationships
- **Query** — Interactive query builder with common patterns
- **Traverse** — Multi-hop graph traversal with chained operations  
- **Visualize** — Interactive graph visualization powered by CogDB

## Example Datasets

Load pre-built examples with one click:
- **Social Network** — Users following each other
- **Movies** — Films, directors, and actors
- **Knowledge Base** — Programming concepts
